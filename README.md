

# 🎓 Multi-Page College Club Website

A simple **multi-page College Club website** built using **HTML** and **Basic CSS**.
This project demonstrates the proper use of **semantic HTML**, **CSS styling**, **Float**, **Flexbox**, **HTML forms**, and clean folder structure.

---

## 📂 Project Structure

```
📁 College-Club-Website
│── index.html        # Home Page
│── about.html        # About Club Page
│── events.html       # Events Page
│── join.html         # Join Club Form Page
│── style.css         # Common Stylesheet
│── images/           # Image Assets
```

---

## 🌐 Pages Included

### **1️⃣ Home Page (index.html)**

* Displays the club name/logo
* Includes navigation to all pages
* Basic intro to the website

---

### **2️⃣ About Page (about.html)**

* Contains **4–5 lines description** of the club
* One relevant club image
* Styled using basic CSS

---

### **3️⃣ Events Page (events.html)**

* Contains **minimum 3 event cards**
* Each card includes:
  ✔ Event Title
  ✔ Short Description
  ✔ Background Color
  ✔ Border
* Events are aligned using **Flexbox**

---

### **4️⃣ Join Club Form (join.html)**

A fully styled HTML form with fields:

* Name
* Branch
* Year
* Email
* Submit Button

This form demonstrates proper HTML form structure and CSS layout.

---

## 🧭 Common Navigation Bar

All pages include a **consistent navigation bar** with links:

* Home
* About
* Events
* Join

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3 (Float + Flexbox)**
* **Basic UI styling**
* **Proper folder structure**
